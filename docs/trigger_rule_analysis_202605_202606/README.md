## 7. 止盈效果口径与玩法对比（2026-07-01 增补）

**本节是最重要的口径修正，详见 `06_止盈效果口径与玩法对比.md`。** 回答用户问题：当前用高倍玩法（中奖有时只回本不算盈利），换网盘或2.0会更好吗？

### 7.1 口径修正
- "止盈效果"不是看命中率回归均值，而是看**触发后开周期能否真正达到+20止盈**
- 早期用 `hit_flag` 模拟止盈率几乎100%是错的——**必须用真实开奖+回本判定**
- 高倍combo命中里 **35% 是回本（对子/顺子/豹子/13/14）不算盈利**，加入回本判定后止盈率从~100%降到39%

### 7.2 三玩法等效净赔（2万期真实开奖计算）

| 指标 | 高倍 | 网盘 | 2.0 |
|---|---|---|---|
| big_small | 等效1.016（22.5%回本）| 0.998（无回本）| 0.985 |
| combo | 等效3.46（35%回本）| 3.32（无回本）| 2.15 |

### 7.3 三玩法止盈率（combo，高命触发，+20/-20/200期）

| 方案 | 高倍 | 网盘 | 2.0 |
|---|---|---|---|
| 组合马尔可夫V1 | 39% | **95%** | 15% |
| 频次趋势V1 | 33% | **97%** | 0% |
| 找平衡 | 47% | **94%** | 22% |
| 小六壬 | 64% | **95%** | 23% |

**combo用网盘止盈率从~50%提升到~90%**，因网盘无回本、方差小、稳定累积达止盈。

### 7.4 玩法推荐
- **big_small / odd_even → 高倍**（回本占比低22.5%，高赔率2.846优势保留，等效1.43 > 网盘0.998）
- **combo → 网盘**（高倍回本35%方差大止盈率低，网盘无回本止盈率95%）
- **2.0 → 不推荐**（combo负期望-0.276，整体最差）

### 7.5 落地建议
4个combo订阅（找平衡/追长龙/频次趋势V1/遗漏回补V1）的 `settlement_rule_id` 从 `pc28_high_regular` 改为 `pc28_netdisk_regular`，预期combo周期止盈率从~50%提升到~90%、止损率从~50%降到~5%。bs/oe订阅（小六壬、组合马尔可夫V1）保持高倍。

> 注：需向平台确认2.0玩法对子/顺子/豹子是否回本（影响2.0 combo期望：仅13/14回本则-0.009近零，都回本则-0.276负）。

## 1. 核心结论（TL;DR）

1. **6/22 把规则1阈值从 `combo<20/bs<40/oe<40` 收紧到 `combo<16/bs<38/oe<38` 是正确的**：
   规则1整体 ROI 从 +0.6% → +3.2%，胜率从 22.6% → 33.3%。来源回测印证收紧后触发更精准、未来命中率不降反升。

2. **没有任何方案具有真实预测优势**：所有方案命中率均值都锚定在 combo≈25%、bs/oe≈50%，周标准差仅 0.4–2.9%，5月与6月均值偏离基本在 ±1.5% 内。
   → **盈亏本质上靠"命中率 vs 盈亏线的安全垫"被赔率杠杆放大**，而非靠方案"变准"。

3. **稳定性由"安全垫厚度 + 规则与指标特性是否匹配"决定，与方案名气无关**：
   - **bs/oe 安全垫厚**（命中50% vs 盈亏线35.1%，垫 +15%）：走「高命」规则时触发后回归均值仍 > 盈亏线，下侧风险低（跌破盈亏线占比仅 0–6%）→ **稳定盈利**
   - **combo 安全垫薄**（命中25% vs 盈亏线15.4%，垫 +9.6%）：周波动 2–3% 就吃掉一半垫 → ROI 月度剧烈波动、易反转 → **不稳定**
   - **combo 不具备回补属性**：连 miss 3–16+ 期后下期命中率仍稳定 23–26%，不回弹 →「低命」规则的回补假设在 combo 上**不成立**

4. **不要根据单月表现调方案**（会追涨杀跌）：找平衡6月差、追长龙6月好，下月都可能反转。**应按"安全垫厚度 + 规则指标匹配"固化配置。**

## 3. 方案分级（按"可被规则稳定利用"的程度）

### A级：稳定可信赖（两规则/两月均稳定盈利）
- **频次趋势V1**：低命oe、高命(bs/oe/combo) 全部稳定盈利。是表现最稳的方案。
- **组合马尔可夫V1 的 odd_even + combo（仅高命）**：高命oe、高命combo 稳定盈利。
- **小六壬预测 高命**：6月 combo+22%、oe+11%；5月 bs+8%。整体偏稳。

### B级：仅特定规则可用
- **组合马尔可夫V1 低命 combo**：稳定**亏损**（5月-9.1%/6月-18.6%），**必须从低命规则移除**。
- **找平衡 低命 odd_even**：两月小盈，可保留但非主力。
- **追长龙 低命 combo**：6月+43.5%但5月-6.2%，**高波动博弈型**，非稳定策略，可小仓位保留。

### C级：不建议/需移出
- **找平衡 高命**：bs 5月+0.4%→6月-21.9%，oe 5月+4.2%→6月-9.4%，两指标都恶化，**移出规则2**。
- **遗漏回补V1**：5/25才上线，仅6月数据，combo高命+137%但样本极小(5注)、不可信；oe高命-2.1%。**待观察，暂不重仓**。

## 5. 重要提醒

- **本报告所有"盈利"结论都建立在 `pc28_high_regular` 结算规则的高赔率上**（bs/oe 赔2.846、combo 赔约6.5）。若切换到 `pc28_netdisk_regular`（bs/oe 赔1.98、combo 赔3.6），盈亏线会从 35.1%/15.4% 抬升到 50.3%/27.8%，**绝大多数组合将转为亏损**——届时需重新评估。
