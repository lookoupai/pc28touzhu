# 06 · 止盈效果口径 + 玩法对比（高倍 vs 网盘 vs 2.0）

> 本篇修正了前几篇的口径问题，并回答用户核心问题：
> 1. **"止盈效果"不是看命中率回归均值，而是看触发后能否真正达到+20止盈**
> 2. **当前用的是PC28高倍玩法，有些中奖只回本不算盈利——如果换网盘或2.0玩法效果会更好吗？**
>
> ⚠️ **2026-07-01 二次修正**：初版曾因把"赔率"误当"净赔"使用，得出"网盘combo止盈率95%"的错误结论，已撤回。本篇为正确净赔口径的最终版。

## 6.1 口径修正：止盈率 ≠ 命中率回归

前几篇用"触发后40期命中率 > 盈亏线"判断正期望，但这是必要条件不是充分条件。
真正衡量"止盈效果"的指标是：**触发后开周期，在+20止盈/-20止损/200期封顶下，达到止盈的概率**。

用来源数据（关联 `lottery_draws` 取真实开奖 triplet，按三种玩法结算规则逐期模拟）回测。

### 必须用真实开奖+回本判定，不能只用 hit_flag

用 `hit_combo=1` 命中 +5.5、miss -1 模拟，止盈率几乎100%——这是错的。
真实高倍玩法中，combo命中里有 **35% 是回本（对子/顺子/豹子/13/14）不算盈利**：
```
组合马尔可夫V1 combo命中6171次: 回本2979(48%) 真赔付3178(52%)
```
加入回本判定后，combo止盈率从"几乎100%"降到 ~40-65%。

### 实盘触发后命中率 ≠ 来源全局命中率

| 组合马尔可夫V1 combo | 命中率 | vs 盈亏线15.4% |
|---|---|---|
| 来源全局 | 25.6% | 正期望 |
| 来源"低命触发后40期"回测 | 27.0% | 正期望 |
| **实盘5月触发后** | **13.5%** | **负期望** |
| **实盘6月触发后** | **13.2%** | **负期望** |

实盘两月稳定13% < 盈亏线15.4%，是**结构性负期望**。根因：低命规则在combo冷区触发=方案该时段预测combo不准（实盘6月组合马尔可夫V1 combo预测高度集中"大双"53%，各值命中率全<15%）。

## 6.4 最终玩法结论

**高倍玩法整体最好**，无论 combo 还是 bs/oe：
- combo：高倍止盈率多数略优于网盘/2.0（赔率高，正期望时段收益大）
- bs/oe：高倍明显优于网盘/2.0（回本占比低22.5%，高赔率2.846优势保留）
- 2.0：因13/14回本吃掉部分收益，整体略差于高倍；combo因13/14回本期望负

### 撤回初版错误建议
初版曾建议"combo改用网盘"——那是基于把赔率误当净赔的错误口径，止盈率虚高到95%。**正确口径下网盘combo止盈率（49%）反而低于高倍（64%）。撤回该建议，combo继续用高倍。**

## 6.6 关于"按指标分流结算规则"的路由能力

用户问：路由里的"模板-消息模板"能否做到按指标用不同赔率？

**结论：消息模板不能解决赔率分流，但当前也不需要分流（高倍全指标最优）。**

原因：
- **消息模板**（`message_templates` 表）只控制"下注文本格式"（如 `{{bet_value}}{{amount}}` → `大单10`），是发给投注端的文字模板，**与赔率/结算规则无关**。
- **结算规则**（`settlement_rule_id`）决定赔率和回本，是**订阅级**配置（`user_subscriptions.strategy_json`），整个订阅所有指标共用一个。
- 当前一个订阅只投一个指标（`play_filter` 单指标），所以"订阅级结算规则"=该指标的结算规则，**不需要分流**，直接给 combo 订阅设高倍即可。
- 真正需要分流的场景是"一个订阅同时投 combo 和 bs"——但代码 `profit_rule_id`+`odds_profile` 是订阅级统一，不支持同订阅内 combo 用高倍、bs 用网盘。届时需拆成两个订阅。

## 6.7 真正能提升止盈效果的方向（回到规则本身）

既然玩法已是高倍（最优），提升止盈效果的杠杆回到**规则配置**（见 `05_结论与配置建议.md`）：

1. **combo 只走高命，不走低命**：实盘 combo 低命触发后命中率13% < 盈亏线15.4%（结构性负期望），高命触发后命中率19-23% > 盈亏线。
2. **combo 高命阈值从35放宽到30**：触发频次×5，止盈率几乎不变（combo无动量，阈值只决定频次）。
3. **bs/oe 走高命**：安全垫厚（50% vs 35.1%），触发后回归均值仍正期望。
4. **移除实盘两月负期望的组合**：组合马尔可夫V1 combo低命、找平衡高命。
5. **止损 -20 保持**（-5会大幅降止盈率，-15/-20止盈率几乎相同）。
