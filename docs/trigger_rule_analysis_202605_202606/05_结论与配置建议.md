## 5.3 方案分级总结

### A级（稳定可信赖，主力）
- **频次趋势V1**：两规则、两月、全指标基本都盈利，是最稳的方案。
- **组合马尔可夫V1（高命 combo+oe）**：高命 combo 与 oe 两月稳定盈利。
- **小六壬预测（高命）**：6月 combo+22%、oe+11%，整体偏稳。

### B级（仅特定规则/小仓位）
- **找平衡（低命 odd_even）**：两月小盈，可保留但非主力。
- **追长龙（低命 combo）**：6月+43.5%但5月-6.2%，高波动博弈，小仓位保留。

### C级（移除/待观察）
- **组合马尔可夫V1 低命 combo**：稳定亏损，必须移除。
- **找平衡 高命**：bs/oe 两月都恶化，移出规则2。
- **遗漏回补V1**：仅6月数据，combo高命+137%是5注样本不可信，待观察。

## 5.4 给操作的核心提醒

1. **不要按单月表现调方案**（追涨杀跌）：找平衡6月差就砍、追长龙6月好就加，下月大概率反转。应按"安全垫厚度 + 规则指标匹配"固化配置。

2. **本结论依赖 `pc28_high_regular` 高赔结算**。若切换到 `pc28_netdisk_regular`（bs/oe赔1.98→盈亏线50.3%，combo赔3.6→盈亏线27.8%），盈亏线大幅抬高，**绝大多数组合将转亏**，需重新评估。

3. **6月数据需分段引用**：6/7–6/21 旧阈值、6/22–6/30 新阈值，混用会误判规则1效果。

4. **combo 的"高命"盈利来自赔率而非动量**：触发后命中率回归25%均值，但 25% > 盈亏线15.4% 仍盈利。这是 combo 高命可用的根因，也是 combo 低命（回补假设）不可用的根因——同一个"无预测优势"事实的两面。

5. **5月 sub7（遗漏回补V1）未启用**（5/25才创建），5月对比表中 sub7 行为空属正常。
